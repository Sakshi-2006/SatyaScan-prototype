# SatyaScan-prototype
# Overview

SatyaScan is an AI-powered digital safety tool designed to detect fake or suspicious online profiles.
It analyzes user data, behavior patterns, and risk indicators to provide real-time risk scoring and alerts, helping users stay safe in digital spaces.

# Problem Statement

Fake profiles and online identity misuse are increasing rapidly, leading to:

Cyberstalking
Harassment
Fraud & impersonation
Lack of trust in digital platforms

# Current systems are:

Slow
Manual
Easy to bypass
 Solution

SatyaScan provides a smart, automated system that:

Detects fake profiles instantly
Assigns a risk score (0–100)
Alerts users about suspicious behavior
Provides actionable insights
